# Seguridad Informática: Riesgos de Acceder a Páginas Maliciosas desde la Universidad

## Descripción

Este proyecto tiene como objetivo concientizar a estudiantes, docentes y personal administrativo sobre los riesgos asociados al acceso a páginas web maliciosas desde equipos o redes institucionales.

## ¿Qué son las páginas maliciosas?

Son sitios web diseñados para realizar actividades perjudiciales, tales como:

* Robo de información personal.
* Instalación de virus, troyanos o ransomware.
* Suplantación de identidad (phishing).
* Obtención de contraseñas y datos confidenciales.
* Compromiso de equipos y redes.

## Riesgos para la Universidad

Acceder a páginas maliciosas desde la red universitaria puede provocar:

1. Infección de equipos institucionales.
2. Pérdida o filtración de información académica.
3. Robo de credenciales de estudiantes y profesores.
4. Interrupción de servicios tecnológicos.
5. Daños económicos y reputacionales para la institución.
6. Uso indebido de los recursos de la red universitaria.

## Buenas Prácticas de Seguridad

* Verificar siempre la URL antes de ingresar información.
* Evitar descargar archivos de fuentes desconocidas.
* No abrir enlaces sospechosos recibidos por correo o mensajería.
* Mantener actualizado el sistema operativo y el antivirus.
* Utilizar contraseñas seguras.
* Reportar actividades sospechosas al área de tecnología.

## Conclusión

La seguridad informática es responsabilidad de todos. Evitar el acceso a páginas maliciosas protege la información personal, los recursos tecnológicos y la integridad de la red universitaria.
